# Part A — エンジニア向け（仕様・技術）

## 共通アーキテクチャ

- **形態**：単一HTMLファイル（HTML + CSS + バニラJS）。ビルド・依存パッケージ・外部CDNなし。
- **動作要件**：モダンブラウザのみ。サーバー不要（`file://` でも動作）。
- **状態管理**：UIは素のDOM操作。永続化は環境にあれば `window.storage`（KVS）を使用、無ければセッション限り（graceful degradation）。
- **AI連携**（①のみ必須機能、②は任意）：Anthropic Messages API（`POST /v1/messages`、`claude-sonnet-4`系）。
  - 環境のAI：ヘッダーにキーを付けない。
  - 自分のAPIキー：`x-api-key` ＋ `anthropic-version` ＋ `anthropic-dangerous-direct-browser-access: true`。
  - 失敗時は必ずルール/テンプレ生成へフォールバックし、理由（`HTTP 4xx`等）を表示。
- **エクスポート**：Markdown / PDF / CSV / メモ(.txt) を `Blob` + `<a download>` でローカル保存。
  - **PDF**：ブラウザ単体でPDFバイナリは生成不可のため、「PDFをダウンロード」は印刷向けに整形した `.html`（`@page`指定入り）を保存、「PDFを印刷」は `window.print()` を別ボタンで提供。
  - **CSV**：先頭に BOM(`\uFEFF`) 付与（Excel互換）。クォート対応の自前パーサ/ビルダー。
- **ファイル名**：自動命名は `YYYYMMDDHHMM_名前_種類_テーマ`。任意名も可。禁止文字（`\ / : * ? " < > |` 等）除去、空白→`_`、40字上限のサニタイズ。拡張子は形式ごとに自動付与。
- **セキュリティ**：ユーザー入力をDOM/印刷へ差し込む箇所は `esc()` でHTMLエスケープ（XSS対策）。入力は外部送信なし（AI整文時のテキスト送信を除く）。
- **アクセシビリティ**：装飾アニメは `@media (prefers-reduced-motion: reduce)` で停止。
- **テーマ**：CSSカスタムプロパティに集約。①②はうさうさ系／メルヘン、③はトロピカル海辺の学校（黒板＋国旗カラー＋波・魚アニメ）。

## ツール別の技術メモ

### ① うさうさジェネレーター
- 入力：種類（free/explain/notice/email/summary/bullets/quiz）・トーン・長さ・モード。
- `buildPrompt()` が種類別プロンプトを構築、`ruleDraft()` がAIなしのたたき台。`callAI()` は `data.content` を `type==='text'` で抽出（位置非依存）。
- 「長さ」は文言と `max_tokens`（500/1000/1500）に反映。設定（種類/トーン/長さ/モード）とAPIキーは別キーで保存（キーは任意・既定オフ）。

### ② バハイ・ウサギ
- 4タブ：依頼ジェネレーター／手順書／お渡しメモ／作業ログ。テーマは朝/昼/夜（時刻で自動選択・記憶）。
- 作業ログは予定/実績を0.25h（15分）丸め（`r025/nz/sgn`）。月次は内製SVG折れ線（実線=実績/破線=予定/点線=目標、1か月は水準線・2か月以上で折れ線）。
- お渡しメモは担当タイプ（文系/理系・真面目/適当）で文面を出し分け、英語併記・記入欄に対応。MD/PDF(html)/CSV/txt出力＋自動命名。

### ③ 月末報告書メーカー
- 入力：基本情報＋実績テーブル（項目/値/メモを行で追加）＋サマリー/所感/課題/来月。
- `reportText/MD/CSV/PrintHTML` の4ビルダー。対象月の既定は前月。装飾（波2層・魚3匹・うさぎ）はヘッダー内に閉じ、文字の背面（z-index）で入力を妨げない。

# Part B — 非IT向け（やさしい使い方）

むずかしい設定はありません。**リンクを開いて、入力して、ボタンを押すだけ**です。

## ① うさうさジェネレーター 🐰✨
**お金をかけない文章づくり支援。書きたいこと、うさぎにおまかせ。**
説明文・お知らせ・メール・要約・クイズを、AIあり/なしどちらでも。

1. つくりたい種類を選んで、内容を入力します
2. 「✨生成する」を押すと文章ができます
3. コピー、または Markdown / PDF / CSV / メモ で保存できます

> 💡 AIが使えない場所でも、文章のひな形は必ず出ます。安心して使えます。
